AI for a Cleaner
Planet

GreenScan uses computer vision to automatically detect waste in images — making environmental monitoring faster, smarter, and accessible to everyone.

🎯 Mission
🤖

AI-Powered Detection

Trained YOLOv8 model identifies garbage, waste bags, litter, and trash with real-time accuracy — from a phone photo or webcam frame.

♻️

Smart Waste Guidance

Beyond detection — we tell you what to do. Disposal suggestions mapped per waste class help close the loop from detection to action.

📍

Location-Aware Logging

Each scan is tagged with a location and timestamp. Build a geographic picture of cleanliness over time in your area.

📊

Data-Driven Insights

Admin dashboard shows detection rates, confidence trends, and exportable reports for community and municipal action.

⚙️ How It Works

📤 Input

User uploads an image or captures a live frame via device camera or DroidCam (phone as webcam over WiFi)

🚀 API Call

Frontend sends the image to Flask backend at /detect endpoint via multipart POST

🧠 YOLO Inference

YOLOv8 model runs on the image, drawing bounding boxes around detected waste classes with confidence scores

💾 Database Storage

Result is stored in SQLite — filename, detection status, confidence, timestamp — for audit trail and analytics

📊 Result Display

Frontend shows side-by-side original vs annotated image, confidence bars, disposal suggestions, and location tag
