# Smart Waste Management using AI

This project uses Artificial Intelligence to detect garbage from images and assist in waste management.

## 📌 Features

- Garbage detection using YOLO model
- Image-based detection
- Live camera detection
- Flask API integration
- Simple UI for user interaction

## 🛠 Technologies Used

- Python
- OpenCV
- YOLO (Ultralytics)
- Flask
- NumPy, Pandas

## 📂 Project Structure

- media/ → contains sample images
- weights/ → trained YOLO model
- GarbageDetector.py → image detection
- GarbageDetectorLive.py → live detection
- GarbageDetectorAPI.py → backend API
